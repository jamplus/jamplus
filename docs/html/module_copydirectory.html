Rules

rule CopyDirectory PARENTS : DESTINATION_PATH : SOURCE_PATH [ : SEARCH_STRING ]

Copies the file contents from SOURCE_PATH to DESTINATION_PATH using SEARCH_STRING as the wildcard.

Parameters
PARENTSThe parent target to link the CopyDirectory dependencies to.
DESTINATION_PATHAn absolute or relative path describing the resultant location of the files to be copied. Any non-absolute paths are relative to the current working directory.
SOURCE_PATHAn absolute or relative path describing the source location where files are to be copied from. Any non-absolute paths are relative to the current working directory.
SEARCH_STRING(optional) The search string to be used for matching files from SOURCE_PATH. If not specified, the default is *. A full description of possible values can be found in the section Patterns and Wildcards.
# Copies all files recursively from source/ to destination/. Does not copy the ignore/ directory.
CopyDirectory all : destination : source : **@-ignore/ ;
